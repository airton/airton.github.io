Dimensionar os elementos com base nas dimensões da janela de visualização sem javascript. Utilizando as unidades de viewport vw e vh.

```css
section {
  width: 100vw;
  height: 100vh;
}
```

- **vw**: largura da viewport
- **vh**: altura da viewport
